## 可观测性要求

- 所有日志必须是结构化的（JSON 格式），不含 PII
- 所有服务边界必须有分布式追踪
- 必须暴露的指标：请求速率、延迟直方图、错误计数

## 智能体如何验证可靠性

- 通过 LogQL（或等效工具）查询日志，验证错误模式
- 通过 PromQL（或等效工具）查询指标，验证性能是否达标
- 本地可观测性栈按 worktree 临时创建，任务完成后销毁
