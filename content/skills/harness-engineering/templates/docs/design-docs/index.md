## 状态含义（智能体行为指南）

- **已验证 ✅**：设计与实现一致，测试通过。智能体可以信赖此文档。
- **草稿 📝**：设计尚未实现。智能体据此实现时需注意可能有未确定的细节。
- **过期 ⚠️**：实现已偏离设计。智能体不应信赖此文档的细节，需先更新。

## 如何添加新设计文档

1. 将 `docs/design-docs/_template.md` 复制为 `docs/design-docs/{功能名}.md`
2. 填写所有章节——重点是让其他智能体能理解"为什么这样设计"和"有什么约束"
3. 在上方目录表中添加条目
4. 将状态设为 **草稿 📝**
5. 实现完成且测试通过后，更新为 **已验证 ✅**
