## 智能体必须遵守的安全规则

1. **数据保护**：静态加密与传输加密；日志和错误信息中绝不包含 PII
2. **输入验证**：所有系统边界处必须进行 Schema 验证
3. **依赖安全**：CI 中自动化 CVE 扫描；禁止 `eval()`、禁止动态 `require()`/`import()`
4. **密钥管理**：密钥仅通过环境变量传递，绝不写入代码

## 安全审查清单

- [ ] 所有用户输入已通过 Schema 验证
- [ ] 无硬编码密钥或凭证
- [ ] 错误信息不泄露内部实现细节
- [ ] 新增的 API 端点已配置认证和授权
- [ ] 新增的依赖已通过 CVE 扫描
